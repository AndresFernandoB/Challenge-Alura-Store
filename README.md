# Análisis de tiendas cadena Alura Store

###Introducción

El informe busca apoyar al Sr. Juan en la decisión de vender una tienda de la cadena Alura Store, utilizando datos de ventas, rendimiento y reseñas de cuatro tiendas. El objetivo es identificar la tienda menos eficiente y presentar una recomendación informada para su venta.

- Propósito: Determinar la tienda con menor desempeño integral.

- Alcance: Ventas, costos de envío, satisfacción del cliente y distribución por categorías.

- Resultado esperado: Recomendación clara, sustentada en indicadores y visualizaciones.

### Datos y estructura

Los dataframes contienen información de ventas y contexto transaccional para cuatro tiendas.

- Producto y categoría: Artículos vendidos y su clasificación.

- Precio y envío: Monto de venta y costos logísticos.

- Fecha y ubicación: Momento y lugar de compra.

- Evaluación de compra: Reseñas y calificaciones de clientes.

- Tipo de pago y cuotas: Métodos utilizados.

- Coordenadas geográficas: Punto de transacción.

Estructura del repositorio:

- Código: AluraStoreLatam.ipynb con el análisis y generación de gráficas.

- Documentación: README.md con resumen de reproducción.

- Imágenes: Carpeta img/ con visualizaciones exportadas.

- Datos: Carpeta base-de-datos-challenge1-latam con CSV de las cuatro tiendas.
